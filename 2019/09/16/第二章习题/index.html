第二章习题
发布于 2019-09-16   更新于 2019-09-16 计算机网络 习题

复习题

R1

R1：列出五种非专用的因特网应用及所用的应用层协议。

应用 协议
Web HTTP
电子邮件 SMTP
流媒体 HTTP
远程终端访问 Telnet
文件传输 FTP

R2

R2：网络体系结构与应用程序体系结构有什么区别？

网络体系结构：7 层 OSI 模型，5 层因特网体系：应用层、运输层、网络层、数据链路层、物理层。

应用程序体系结构是由应用程序研发者设计并规定如何在各种端系统组织该程序。常见的结构有 c/s 和 b/s 结构。

R3

对两进程之间的通信会话而言，哪个为客户，哪个为服务器？

发起会话的是客户，接受会话的是服务器。

R4

对于一个 p2p ，你同意一个通信会话不存在客户端和服务器端的概念吗？

不同意。即使在 p2p 中，也能够在一条连接中找到一个服务的请求方和接收方，而这就可以分出客户端和服务器端。

R8

列出一个运输协议能够提供的 4 种宽泛类型的服务。对于每种服务类型，指出是 UDP 还是 TCP。

服务 类型
可靠数据传输 TCP
吞吐量 无
定时 TCP
安全性 无

R10

握手协议的作用是什么？

客户向服务器发送一个报文段，服务器用一个报文段做出确定和响应，最后客户向服务器返回确认。

建立 tcp 连接，传输数据。

R11

为什么 HTTP SMTP POP3 运行在 TCP ？

TCP 提供可靠数据传输，能够保证数据能够没有错误的，按顺序的从发送方到接收方。

R12

考虑一个电子商务网站需要爆流每一个客户的购买记录，简述如何使用cookie 完成该功能。

首先 cookie 储存在请求报文的 cookie 首部行，在响应报文中，在用户的端系统中有一个 cookie 文件，在网站的后端数据库中。
